# Statically typed Language

  • A type of language where we need to define type of variable before using it.
  • We need to declare data type of value which we are going to store in variable.

Syntax:-

data_type variable_name = values;

Basic code:-

  • public class First{
      public static void main(String[] Saurabh){
        // Data type is written before which type of value is going to store.
        int age = 23;
        System.out.println(age);
      }
    }
